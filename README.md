# STAT184-Homework-4.4

## About This Project
This project provides contains the files for a reproducible data analysis completed using Quarto and R. The project contains three main analyses: a comparison of the busiest airports in the world, a Monte Carlo integration, and a planning excercise using generative AI. 

## Data Source
This project used data from two different sources. The airport data was scraped from the following wikipedia page: https://en.wikipedia.org/wiki/List_of_busiest_airports_by_passenger_traffic. This data set shows a list of the busiest airports across the globe with attributes such as Country and Total Passengers. The calcium dataset, 'calcium.csv', was a calcium database provided as part of course materials. This database tracks the calcium levels in two groups of females over time.
